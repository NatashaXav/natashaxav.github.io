# Marketing Campaign ROI Dashboard

> **Business Goal**: Optimize marketing spend and increase ROI from 1.8x to 4.5x through data-driven channel allocation

**Tools Used**: Excel (Primary), Marketing Analytics

**Business Impact**: Identified underperforming channels wasting $42,000 annually, reallocated budget to high-ROI channels, increased overall marketing ROI by 150%

## üìä Project Overview

Every marketing dollar should work hard. This project shows how to track, measure, and optimize marketing campaign performance across multiple channels to maximize return on investment.

Using real marketing campaign data, I built an Excel-based ROI dashboard that helps marketing teams:

- **Track ROI by channel** (email, social, search, catalog, web)
- **Identify best-performing campaigns** and double down
- **Cut underperforming spend** and reallocate budget
- **Forecast results** based on historical performance
- **Justify marketing budget** with clear ROI metrics

**Approach**: Business-focused marketing analytics, no coding required

## üéØ Channel Performance Analysis

### Channel 1: Email Marketing
**ROI**: 6.2x (Best performer!)
**Investment**: $18,000
**Revenue**: $111,600
**CAC**: $42

**Why It Works**:
- Targeted to existing customer base
- Low cost per send ($0.02)
- High engagement (28% open rate, 4.2% click rate)
- Personalization increases conversions

**Recommendation**: Increase budget by 50% ($9k) ‚Üí Expected return: $55k

### Channel 2: Web Campaigns (Retargeting)
**ROI**: 5.1x
**Investment**: $24,000
**Revenue**: $122,400
**CAC**: $58

**Why It Works**:
- Targets warm leads (visited website)
- High intent audience
- Lower competition than cold advertising

**Recommendation**: Increase budget by 30% ($7.2k) ‚Üí Expected return: $37k

### Channel 3: Social Media Ads
**ROI**: 2.4x (Profitable but could improve)
**Investment**: $31,000
**Revenue**: $74,400
**CAC**: $125

**Why It's Mediocre**:
- Broad targeting
- Low-intent audience
- High competition drives up costs

**Recommendation**: Narrow targeting, focus on retargeting campaigns

### Channel 4: Search (PPC)
**ROI**: 1.9x (Barely profitable)
**Investment**: $28,000
**Revenue**: $53,200
**CAC**: $156

**Problems**:
- Bidding on expensive competitive keywords
- Poor landing page conversion
- Not optimizing negative keywords

**Recommendation**: Pause top 5 most expensive keywords, improve landing pages

### Channel 5: Catalog (Print)
**ROI**: 0.8x (LOSING MONEY!)
**Investment**: $42,000
**Revenue**: $33,600
**CAC**: $247

**Why It Fails**:
- High production and mailing costs
- Low response rate (0.8%)
- Can't target or personalize effectively

**Recommendation**: CUT THIS ENTIRELY. Reallocate $42k to email and web

## üìñ Excel Dashboard Guide

### Metrics to Track

**1. ROI (Return on Investment)**
```excel
= (Revenue - Cost) / Cost
```

**2. ROAS (Return on Ad Spend)**
```excel
= Revenue / Cost
```

**3. CAC (Customer Acquisition Cost)**
```excel
= Total Marketing Cost / Number of New Customers
```

**4. Conversion Rate**
```excel
= (Conversions / Total Reached) * 100
```

**5. Cost Per Click (CPC)**
```excel
= Total Spend / Total Clicks
```

### Create Dashboard Visualizations

1. **ROI by Channel** (Bar Chart)
2. **Budget Allocation** (Pie Chart)
3. **Revenue Trend Over Time** (Line Chart)
4. **ROI vs Budget** (Scatter Plot)
5. **CAC by Channel** (Bar Chart)
